# XatBot Talent 2026 - Edgar Alba

**XatBot intel·ligent per WordPress** que utilitza **Google Gemini** per respondre preguntes sobre el teu contingut web de forma dinàmica.

---

## ✨ Característiques

- Scraping en temps real de la teva pàgina web
- Respostes en català naturals i professionals
- Widget flotant fàcil d'incrustar a WordPress
- Execució des de Google Colab amb Ngrok
- Respostes limitades i netes (sense Markdown)

---

## 🚀 Com utilitzar

### 1. Google Colab
1. Obre el fitxer `XatBot_talent_2026.ipynb`
2. Configura els **Secrets**:
   - `GEMINI_API_KEY`
   - `NGROK_AUTH_TOKEN`
3. Executa les cel·les
4. Copia la URL de Ngrok

### 2. WordPress
1. Copia el contingut de `Widget_talent.html`
2. Enganxa'l en un bloc **HTML personalitzat**
3. Actualitza la URL del Ngrok
