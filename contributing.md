# Com contribuir al XatBot Talent 2026

Gràcies per voler contribuir al projecte!

## Com contribuir

1. Fes **Fork** del repositori
2. Crea una branca per la teva modificació (`git checkout -b millora/nom`)
3. Fes els canvis
4. Fes commit amb missatges clars
5. Obre una **Pull Request**

## Àrees on pots ajudar

- Millores al Widget HTML/CSS
- Millores en el scraping (més contingut, millor extracció)
- Optimització de prompts per Gemini
- Gestió de respostes més naturals
- Documentació
- Correcció d'errors
